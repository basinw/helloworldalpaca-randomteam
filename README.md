# HelloWorld #Alpaca - Random System
## Command for Development Server
### Install Dependencies !
```
yarn
```
### Run FrontEnt
```
yarn start
```
### Run Backend
```
nodemon server
```

## Command for Deploy
### Install Dependencies !
```
yarn or npm install
```
### Build FrontEnd !
```
yarn build or npm run build
```
### Add Another command with PM2!
```
node server.js
```
